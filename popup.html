GCal Event Multiselect




Instructions
  1. Set your calendar ID: The calendar ID for your primary calendar (the one titled your name) is just your gmail address. If you are using a different calendar, you can find its calendar ID in the calendar settings.
  2. Select the events to move: CTRL/⌘ + CLICK to select individual (non "All day") events. If you have a block of back-to-back events you'd like to move, select the first event in the block, then SHIFT + CLICK the last event to select all the events in between as well.
    • Move the events: Move one of the selected events, then press CTRL/⌘ + ENTER to move the rest.
    • Delete the events: Move one of the selected events, then press DEL to move the rest.
